Euclid's Algorithm

Euclid's Algorithm, abbreviated EucAlg , is an algorithm for calculating the gcd of two positive integers. It does so recursively with an algorithmic complexity of O(log n). Or, as Wikipedia describes,

In mathematics, the Euclidean Algorithm, or Euclid's algorithm, is an efficient method for computing the greatest common divisor (GCD) of two integers, also known as the greatest common factor (GCF) or highes common factor (HCF). Also, you can use italics

Euclid's algorithm proves fruitful in the implementation of RSA cryptography. Here is some psuedo code for it's implementation in python. 

 
    def EucAlg(a, b):
        if (b == 0):
            return a
        else:
            return EucAlg(b, a%b)

Clearly, it is potent algorithm that can be implemented rather briefly.
